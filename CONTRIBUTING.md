# Contributing

Thanks for contributing to `quentin`.

## Local setup

```bash
npm install
npm run lint
npm run typecheck
npm test
```

## Pull requests

Please include:

- What changed
- Why it changed
- How you tested it

## Commit messages

Use Conventional Commits:

- https://www.conventionalcommits.org/en/v1.0.0/

A Husky + Commitlint hook checks commit message format.
